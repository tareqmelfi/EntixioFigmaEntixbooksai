## نظرة عامة

**Entix Books** هو نظام محاسبة مالية سحابي متكامل مصمم خصيصاً للسوق السعودي والأمريكي. يوفر النظام حلاً شاملاً للشركات الصغيرة والمتوسطة مع دعم كامل للامتثال لمتطلبات هيئة الزكاة والضريبة والجمارك (ZATCA) للفوترة الإلكترونية.

## الأدوار المحددة مسبقاً (7 أدوار)

1. **المالك (Owner)**: صلاحيات كاملة + إدارة الفوترة
2. **المدير (Admin)**: صلاحيات كاملة ما عدا الفوترة
3. **المحاسب (Accountant)**: المحاسبة والفواتير والتقارير
4. **الموظف (Employee)**: محدود - السجلات الشخصية فقط
5. **العميل (Customer)**: بوابة العملاء - عرض الفواتير والدفع
6. **المورد (Supplier)**: بوابة الموردين - عرض أوامر الشراء
7. **المساهم (Shareholder)**: بوابة المساهمين - عرض التقارير المالية

## التكاملات الخارجية

- ✅ **ZATCA FATOORAH** - الفوترة الإلكترونية السعودية (متصل)
- ⏳ **Stripe Payments** - معالجة المدفوعات (قريباً)
- ⏳ **STC Pay / Mada** - المدفوعات المحلية السعودية (المرحلة 2)
- ⏳ **Google Vision OCR** - قراءة الفواتير تلقائياً (المرحلة 3)
